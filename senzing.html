senzing.g2config module

The g2config package is used to modify the in-memory representation of a Senzing configuration. It is a wrapper over Senzing’s G2Config C binding. It conforms to the interface specified in g2config_abstract.py

To use g2config, the LD_LIBRARY_PATH environment variable must include a path to Senzing’s libraries.

Example:

export LD_LIBRARY_PATH=/opt/senzing/g2/lib

senzing.g2product module

The g2product package is used to inspect the Senzing product. It is a wrapper over Senzing’s G2Product C binding. It conforms to the interface specified in g2product_abstract.py

To use g2product, the LD_LIBRARY_PATH environment variable must include a path to Senzing’s libraries.

Example:

export LD_LIBRARY_PATH=/opt/senzing/g2/lib
